# My First Project

مرحبًا! هذا هو أول مشروعي باستخدام HTML، صُمم بعناية لإظهار فهمي الأساسي لتصميم صفحات الويب.  

**حول المشروع:**  
- المشروع مبني بالكامل بـ HTML، ويُظهر مهاراتي في هيكلة صفحات الويب.  
- يوضح المشروع اهتمامي بتعلم البرمجة وابتكاري في تنفيذ أفكار بسيطة لكنها عملية.  

**لماذا هذا المشروع مميز:**  
- تجربة أولى حقيقية تُظهر قدرتي على تحويل المعرفة النظرية إلى كود عملي.  
- مناسب للـHR للتعرف على مستوى مهاراتي في HTML وتنظيم المشاريع.  

**معلومات شخصية:**  
- الاسم: محمود حمدي  
- العمر: 16 سنة  
- الشغف: تعلم البرمجة وتطوير الويب
